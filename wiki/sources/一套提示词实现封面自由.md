# 一套提示词帮你实现小红书、公众号封面自由

> 歸藏分享的模块化封面生成提示词方案，用 HTML+CSS 代码方式一键生成小红书和公众号封面，DeepSeek V3 0324 也可实现接近 Claude 3.7 的效果。

## 核心内容

作者之前做过网页生成提示词，用类似思路解决自媒体封面难题（小红书 & 公众号封面），花两天搞定这套**模块化设计**的一键生成封面提示词。

### 提示词四模块架构（[重点/高亮]）

两套提示词（公众号 / 小红书）由四个部分组成：

1. **角色设定（绿色）**：设定 AI 角色为优秀网页和营销视觉设计师，具有丰富 UI/UX 经验，擅长现代设计趋势与营销策略融合。
2. **基本要求（黄色）**：约束画面比例和排版（与风格无关），总结优质封面共性。
   - 公众号：整体比例 3.35:1，左边 2.35:1 主封面 + 右边 1:1 朋友圈分享封面，朋友圈封面只需四个大字铺满（上二下二），文字占至少 70% 空间，两封面共享背景色和点缀装饰。
   - 小红书：比例 3:4（宽:高），无边框 div 画布直角卡片，文案提炼 30-40 字精华，3-4 种字号层次感，主标题比副标题大三倍以上，主标题 2-3 关键词特殊处理（描边/高亮/变色）。
3. **风格要求（紫色）**：两套提示词的精华部分，**风格提示词两个封面共用**，后续 YouTube 封面等也可复用。
4. **用户输入内容（红色）**：标题/文案，可不按字数输入（AI 帮优化），背景图/素材图链接也放这里。

### 技术实现方式

- 使用纯 HTML + CSS 编写（公众号版），Tailwind CSS CDN 优化比例/样式。
- 小红书版增加 JavaScript + html2canvas 实现一键保存为图片。
- 引用 Google Fonts、Font Awesome 等在线资源。
- 响应式设计，代码可在现代浏览器直接运行。
- 最下方增加图片下载按钮。

### 如何探索风格

作者方法：将自己觉得不错的排版图片喂给 AI，让它总结风格描述（而非让 AI 自行发散）。

风格生成提示词：`总结这几个小红书封面的风格，为他起一个名字，而且分设计风格、文字排版风格、视觉元素风格分开描述。`

### 十套精选风格提示词

每套风格均从「设计风格 / 文字排版风格 / 视觉元素风格」三维度描述：

1. **柔和科技卡片风**：圆角卡片 + 淡紫/浅黄/粉色柔和色调 + 极简留白 + 阴影微立体 + 数据大字号突显。
2. **现代商务资讯卡片风**：深绿/深红色调 + 大面积单色背景 + 三级信息层级 + 网格底纹肌理。
3. **流动科技蓝风格**：蓝白渐变 + 流线型曲线 + 圆角矩形 + 无衬线字体 + 半透明蓝色波纹/螺旋。
4. **极简格栅主义封面风格**：黑白极简 + 网格排版 + 中英文混排装饰 + 摄影与排版结合。
5. **数字极简票券风**：类登机牌/门票结构 + 中英混排 + 多向排列 + 符号化装饰 + 手写风点缀。
6. **新构成主义教学风**：黑红白三色 + 学术实验美学 + 日式现代主义 + 极端对比字阶 + 红线引导。
7. **奢华自然意境风**：暗调景观 + 意境式呈现 + 悬浮式标题 + 摄影级光影 + 东方水墨意境。
8. **新潮工业反叛风**：黑底 + 荧光黄 + 巨型中文标题 + 轮廓线英文 + 拆分重组文本 + 地下文化气质。
9. **软萌知识卡片风**：粉色/米黄/淡紫 + 大圆角 + Q 版表情角色 + 口语化表达 + 感叹号/emoji 点缀。
10. **商务简约信息卡片风**：浅灰/白背景 + 高对比度 + 问答式标题 + 色块编号 + 产品实物展示。
