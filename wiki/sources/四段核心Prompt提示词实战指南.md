# 四段核心Prompt提示词实战指南

## 来源信息
- 物理文件：[[raw/4 段 Prompt 提示词，生成PPT、生成3D动画，生成网站，生成万物....md]]
- 作者：向阳乔木 / 歸藏

## 核心要点
1. **角色赋能与约束设定**：从前端工程和专业设计师视角定义模型人设，严格限定输出格式和模块结构。
2. **结构化输出约束**：通过 XML 或 Markdown 标记规范，使 LLM 能够直接生成符合下游渲染引擎解析标准的结构化数据。
3. **通用万物生成逻辑**：将复杂任务拆解为阶段化思考链条（CoT），引导模型先生成大纲骨架再填补细节内容。
