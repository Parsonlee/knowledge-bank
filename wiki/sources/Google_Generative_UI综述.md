# Google Generative UI 综述

## 来源信息
- 物理文件：[[raw/Generative UI_ A rich, custom, visual in....md]]
- 来源：Google Research Blog

## 核心要点
1. **动态响应交互需求**：打破传统预渲染 Web 页面界限，根据用户的 Prompt 实时生成最适宜当前任务的定制交互界面组件。
2. **模型与组件渲染绑定**：通过声明式组件协议与前端框架整合，使 AI 能够智能调配图表、表单与布局容器。
3. **下一代应用体验方向**：为 AI-Native 应用提供了从纯文字聊天迈向全维度视觉交互体验的技术基石。
