# 让 AI Agent 实时生成个性化交互式可视化体验

## 来源信息
- 物理文件：[[raw/让 AI Agent 实时生成个性化交互式可视化体验.md]]
- 平台：微信公众号

## 核心要点
1. **定义设计规范 (Design System)**：由设计师提前确定视觉风格，提取关键的 Design Tokens（设计变量：主色调、功能色、圆角大小、阴影等），确保 Agent 生成界面风格统一。
2. **自动化代码模板渲染**：通过将设计系统与 LLM 提示工程结合，构建高复用性的前端组件库，供 [[实体_Claude]] 等编程模型实时调用渲染。
3. **交互式个性化生成**：用户通过自然语言输入意图，AI Agent 解析需求后动态拼装并呈现前端可视化交互体验，替代传统的静态图表展示。
