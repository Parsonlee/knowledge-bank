## 核心要点

1. **图像理解三层次**：分类（Classification）→ 检测（Detection）→ 分割（Segmentation，含语义分割与实例分割）。检测任务输出为 (类别, 边界框坐标) 列表。

2. **两阶段检测模型**（Region-based）：
   - **R-CNN**：首次将 CNN 用于区域检测，Selective Search 生成 ~2000 候选区域，逐区域通过 AlexNet 分类。引入 IoU 判定正负样本、Bounding-Box Regression 微调坐标。缺点：三个独立模型、重复计算。
   - **Fast R-CNN**：全图只过一次 CNN 获取共享特征图，RoI Pooling 将不同大小候选区域映射为等长特征向量。多任务损失联合训练分类 + 定位。
   - **Faster R-CNN = RPN + Fast R-CNN**：RPN 替代 Selective Search，在共享特征图上滑窗生成多尺度多宽高比 Anchor，二分类（前景/背景）+ 坐标回归。全流程端到端可训练，约 5fps。

3. **单阶段检测模型**（Region-free）：
   - **YOLO**：将检测视为回归问题。图像分为 S×S 网格，每格预测 B 个框（坐标+置信度）+ C 个类别概率。速度快、全局上下文好，但对小目标和密集目标弱。
   - **SSD**：多尺度特征图检测，每个位置多种尺寸/宽高比 Anchor，逐框预测类别概率。兼顾速度与精度，成为后续单阶段工作的基础。

4. **检测框架通用结构**：Backbone（特征提取）+ Detection Head（分类 + 回归）。两阶段 head 串行（RPN → RCNN），单阶段 head 并行。

## 关键引文

> R-CNN 的两个贡献：证明 CNN 可用于区域定位与分割；有监督预训练 + 领域微调在标注数据稀缺时有效。

> Faster R-CNN = RPN + Fast R-CNN，anchor 机制后被 YOLOv2 等借鉴。
