## 核心要点

### 1. 什么是 AI 智能体

- **定义**：以高度独立性代表用户完成任务的新兴软件范式，LLM 控制工作流执行
- **智能体循环（Agentic Cycle）**：评估状态 → 判断是否调用工具 → 比对目标 → 循环直至退出条件
- **与传统软件区别**：传统软件是"辅助执行"，智能体是"代理执行"
- 集成推理、记忆、工具使用、工作流管理

### 2. 适用场景三类

1. **复杂决策**：需要细微判断力（退款审批综合多因素、欺诈识别微妙模式）
2. **难以维护的规则系统**：将政策文档作为上下文动态应用，而非硬编码
3. **非结构化数据处理**：邮件/PDF/图片等（保险理赔、医疗记录）

### 3. 三大基石：模型 + 工具 + 指令

**模型（LLM）**：
- "从高到低"策略：原型阶段用最强模型建立性能基线，消除"模型能力不足"变量
- 验证可行性后再优化经济性，可使用异构模型部署

**工具三类**：
| 类型 | 描述 | 示例 |
|------|------|------|
| 数据检索 | 只读，获取信息 | query_database、search_knowledge_base |
| 动作执行 | 改变外部系统状态 | send_email、update_crm_record |
| 编排 | 将其他智能体作为工具调用 | triage_support_ticket（调用子 Agent）|

**指令最佳实践**：
- 利用现有 SOP/FAQ 文档转化为步骤化指令
- 明确化与任务分解："首先，询问用户的订单号" 优于 "获取订单信息"
- 使用提示模板 + 参数化变量动态注入 CRM 数据
- 处理边界情况：明确 fallback 逻辑
- 视指令为代码：版本控制 + 同行评审 + 评估测试集

### 4. 编排架构

**原则**：从简到繁，增量演进，避免过早引入复杂多智能体系统。

**单一智能体**：
- 一个 LLM + 工具集 + 指令，运行循环直至退出
- 本质是"有状态的微服务"，维护整个 Run 生命周期内的状态
- 绝大多数初期场景的推荐起点

**多智能体——主管模式（Manager Pattern）**：
- 主管智能体理解总目标，分解子任务委派给下属智能体
- 下属对主管而言是可调用的"工具"
- 适合可并行处理、需综合多种信息的任务
- 类比：项目经理 + 专家团队

**多智能体——去中心化模式（Decentralized Pattern）**：
- 对等智能体之间传递/交接控制权与上下文
- 一个智能体判断后续超出自身能力时移交给更合适的智能体
- 适合线性流水线式、不同阶段专家序贯处理的流程
- 类比：部门间工作交接流程

### 5. 安全护栏：分层防御体系

**原则**：深度防御（Defense in Depth），多层独立检查，任意一层被绕过仍有后续捕获。

| 护栏类型 | 目的 | 实现方法 |
|---------|------|---------|
| 相关性分类器 | 保持在预设功能范围内 | 独立 LLM 分类器 / 关键词主题模型 |
| 安全分类器 | 检测不安全/违规内容 | OpenAI Moderation API / 内部安全模型 |
| PII 过滤器 | 防止个人信息泄露 | Regex 规则 / NER 模型 |
| 工具风险评估 | 高风险操作门控 | 为工具附加 risk_level 元数据，触发人工审批 |
| 输出验证 | 确保格式/语调/合规性 | Pydantic 验证 / LLM 评估品牌指南 |
| 审计 | 记录全部行为日志 | 思考过程 + 工具调用历史 + 最终输出 |

### 6. 人工监督（HITL）机制

**升级触发器三类**：
1. **失败阈值**：连续多次相同错误，自动暂停升级给人类
2. **高风险操作**：大额退款/删除生产数据/修改财务信息 → 默认需人工批准
3. **低置信度/模糊性**：智能体主动识别不确定状态，向用户/操作员请求帮助

**HITL 的战略价值**：不是降级方案，而是建立信任的核心机制；渐进式授权（退款上限从 100 → 500）。

## 关键引文

> "这标志着与传统软件的根本性区别。传统软件，即便是集成了自动化工作流的系统，其本质上是作为用户的工具……而智能体则能够接管整个工作流，并自主地执行。" [重点/高亮]

> "指令即代码（Instructions as Code）……应与应用程序代码一样，被存储在版本控制系统中，接受同行评审，并拥有一套配套的评估测试集。" [重点/高亮]

> "选择错误的编排模式，可能会导致智能体之间通信效率低下、出现流程瓶颈，或者最终无法有效地解决用户的根本问题。" [高亮]
