# 目标检测入门：评测与训练技巧

## 核心要点

1. **[重点/高亮] 多尺度训练/测试**：输入图片尺寸对检测性能影响显著，多尺度是提升精度最明显的技巧之一。基础网络生成比原图小数十倍的特征图，小物体特征难以捕捉。训练时使用更大、更多尺寸图片可提高对物体大小的鲁棒性，仅测试阶段引入多尺度也可获得增益。

2. **评测指标**：
   - Precision/Recall：查准率与查全率的权衡（Type I/II error）
   - IoU（交并比）：判定预测框是否正确的阈值标准
   - AP：单类别在不同 Recall 水平的平均精度
   - mAP：所有类别 AP 的均值，为检测任务核心指标
   - COCO 风格 mAP：IoU 阈值从 0.5 到 0.95 步长 0.05 取均值，另分小/中/大物体 AP
   - FPS：速度指标

3. **基准数据集**：
   - Pascal VOC：20 类，VOC07/12 为经典拆分
   - MS COCO：80 类，像素级实例标注，平均每张图 3.3 类 7.7 实例，强调非 iconic 场景
   - Cityscapes：30 类城市道路场景

4. **训练技巧**：
   - 数据增强：随机翻转/裁剪/噪声，需同步变换标注框
   - 全局语境（Global Context）：将整张图作为一个 RoI 池化后拼接到各候选特征
   - 预测框微调/投票法（Box Refinement）：迭代回归 + NMS 后加权投票
   - OHEM（在线难例挖掘）：按 R-CNN 预测 loss 选最难样本训练
   - Soft NMS：不硬性丢弃高 IoU 重叠框，而是按 IoU 惩罚置信度（高斯/线性衰减）
   - RoIAlign：用双线性插值替代 RoI Pooling 的量化取整，消除对齐误差（Mask R-CNN 提出）
   - 更优 Anchor 先验：k-means 聚类训练集框尺寸（YOLOv2）
