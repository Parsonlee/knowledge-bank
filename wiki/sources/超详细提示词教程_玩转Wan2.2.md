通义万相 2.2（[[实体_通义万相Wan2.2]]）文生视频 Prompt 写作技巧完整教程，系统整理基础/进阶公式及完整影视级美学控制词典。

## Key Points

- **三套 Prompt 公式**（详见 [[概念_文生视频提示词公式]]）
  - 基础公式：主体 + 场景 + 运动（适合新用户/灵感启发）
  - 进阶公式：主体描述 + 场景描述 + 运动描述 + 美学控制 + 风格化
  - 图生视频公式：运动 + 运镜（图像已确定主体/场景/风格）

- **影视级美学控制维度**（详见 [[概念_视频美学控制词典]]）
  - 光源类型（9）：日光/人工光/月光/实用光/火光/荧光/阴天光/混合光/晴天光
  - 光线类型（8）：柔光/硬光/高对比度/侧光/底光/低对比度/边缘光/剪影
  - 时间段（5）：白天/夜晚/日出/日落/黎明
  - 景别（5）：特写/近景/广角/中近景/全景
  - 构图（5）：中心/平衡/左右侧/对称/短边构图
  - 镜头焦段（4）：中焦距/广角/长焦/超广角-鱼眼
  - 镜头类型（3）：干净单人/群像/三人镜头

- **运镜**
  - 基础运镜：镜头推进/拉远/上摇
  - 高级运镜：手持/复合/跟随/环绕运镜

- **风格化（视觉风格）**：像素风格/3D游戏/木偶动画/二次元

- **特效镜头**：移轴摄影/延时拍摄

- **人物情绪**：愤怒/恐惧/高兴；**运动场景**：街舞/跑步/橄榄球

- **核心规律**
  - 提示词描述越完整、精确、丰富，生成视频品质越高、越贴近期望
  - 每个类别均配完整中文示例 prompt，可直接复用或组合
  - 所有视频效果均使用通义万相 2.2 生成

- **体验渠道**：中文官网 tongyi.aliyun.com/wanxiang；英文官网 wan.video
