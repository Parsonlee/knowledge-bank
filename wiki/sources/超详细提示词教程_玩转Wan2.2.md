# 超详细提示词教程：玩转 Wan2.2

> 通义万相 2.2（Wan2.2）官方视频生成提示词教程，系统整理基础与进阶公式及完整词典。

## Key Points

- **模型定位**：Wan2.2（通义万相 2.2）是通义大模型团队推出的视频生成模型，支持文生视频与图生视频两种模式。

- **两套 Prompt 公式**
  - **基础公式**：主体 + 场景 + 运动
  - **进阶公式**：主体描述 + 场景描述 + 运动描述 + 美学控制 + 风格化
  - **图生视频专用**：运动描述 + 镜头运动

- **光源类型（8 种）**：日光、人工光、月光、实景光、火光、荧光灯、阴天漫射光、混合光

- **光线类型（8 种）**：柔光、硬光、高对比光、侧光、底光、低对比光、轮廓光、剪影光

- **时间段（5 种）**：白天、夜晚、日出、日落、黎明

- **景别（5 种）**：特写、中近景、广角景、中景、全景

- **构图方式（5 种）**：中心构图、均衡构图、左右构图、对称构图、短边构图

- **镜头焦距（4 种）**：标准镜头、广角镜头、长焦镜头、超广角鱼眼镜头

- **人物组合（3 种）**：单人清晰、群像、三人组合

- **角色情绪（3 种）**：愤怒、恐惧、喜悦（均附完整中文示例 prompt）

- **运动示例（3 种）**：街舞、跑步、橄榄球运动

- **基础镜头运动（3 种）**：推镜头（push-in）、拉镜头（pull-out）、仰拍上移（tilt-up）

- **进阶镜头运动（4 种）**：手持抖动、复合运动、跟踪镜头、环绕镜头（orbiting）

- **视觉风格（4 种）**：像素风格、3D 游戏风格、木偶动画风格、动漫风格

- **特效（2 种）**：移轴效果（tilt-shift）、延时摄影（timelapse）

- **使用规律**
  - 词典中每个类别均配有完整中文示例 prompt，可直接复用或组合。
  - 进阶公式中美学控制项（光线、景别、构图）对画面质感影响最大。
  - 图生视频时镜头运动描述优先级高于静态构图描述。
