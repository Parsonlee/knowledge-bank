# 梯度下降优化器可视化解释

## 摘要

通过可视化动画直观解释五种梯度下降方法的工作原理与差异对比。原文译自 Lili Jiang 的 A Visual Explanation of Gradient Descent Methods。

## 主要内容

### 原版梯度下降（Vanilla Gradient Descent）

- 目标：最小化损失函数，快速可靠找到全局最小值
- 算法：delta = -learning_rate * gradient; theta += delta
- 缺点：只按梯度方向迭代，易陷入局部极小值/鞍点/高原

### 动量（Momentum）

- 借鉴物理学：无摩擦碗中滚球，积累动量推动前进
- 公式：
  - sum_of_gradient = gradient + previous_sum_of_gradient * decay_rate
  - delta = -learning_rate * sum_of_gradient
  - theta += delta
- decay_rate 通常 0.8-0.9（0=原版GD，1=永不停止）
- **优势**：移动更快（积累动量）；有机会逃脱局部极小值；更好通过高原区

### AdaGrad（Adaptive Gradient）

- 跟踪梯度平方之和，为不同方向调整梯度
- 公式：
  - sum_of_gradient_squared = previous + gradient²
  - delta = -learning_rate * gradient / sqrt(sum_of_gradient_squared)
- 解决稀疏特征训练慢的问题：更新多的特征未来更新少，让稀疏特征赶上
- **优势**：更好避开鞍点（沿对角线方向而非只沿陡峭方向）
- **缺点**：梯度平方和只增不减，后期几乎停止

### RMSProp（Root Mean Square Propagation）

- 添加衰减因子修复 AdaGrad 后期过慢问题
- 公式：
  - sum_of_gradient_squared = previous * decay_rate + gradient² * (1-decay_rate)
  - delta = -learning_rate * gradient / sqrt(sum_of_gradient_squared)
- 只有最近的梯度平方有意义，很久以前的被遗忘
- 比 AdaGrad 更快，梯度平方和保持可控大小

### Adam（Adaptive Moment Estimation）

- 同时兼顾 Momentum 和 RMSProp
- 公式：
  - sum_of_gradient = previous * beta1 + gradient * (1-beta1)【Momentum】
  - sum_of_gradient_squared = previous * beta2 + gradient² * (1-beta2)【RMSProp】
  - delta = -learning_rate * sum_of_gradient / sqrt(sum_of_gradient_squared)
- beta1=0.9（一阶矩），beta2=0.999（二阶矩）
- **速度来自动量 + 适应不同方向的能力来自 RMSProp**
- 深度学习问题的常用选择

### 对比总结

- 一阶矩（梯度之和）：Momentum 用，获得速度
- 二阶矩（梯度平方之和）：AdaGrad/RMSProp/Adam 用，处理稀疏特征/避开鞍点
- AdaGrad 无衰减，RMSProp 有衰减加速
- Adam = 一阶矩 + 二阶矩，通常最好
- 实验：Adam 是唯一能越过局部极小值后再侧向移动找到全局极小值的方法
