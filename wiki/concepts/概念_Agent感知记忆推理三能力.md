# 概念：Agent 感知记忆推理三能力

大模型 Agent 在完成任务的计划-决策-控制循环中，需要三种核心能力：感知、记忆、推理。CoT 从三个维度赋能 Agent。

## 感知（Perception CoT）

- 对环境反馈、人类指令进行理解和意图识别
- CoT 让模型逐步关注接收信息：`"Answer: Let's think step by step. I see..., I need to..."`
- 支持多模态感知：
  - 语言为中心：其他模态转为语言理解
  - 图像为中心：视觉优先的感知
  - 多模态中心：文本+图像统一（数据/计算挑战大）

## 记忆（Memory CoT）

- **短期记忆（工作记忆）**：动态上下文，历史动作链，随交互变化
- **长期记忆**：静态历史知识，可存于可训练参数或外部记忆库
- 长序列时线性链效率下降，两种优化方式：
  - **树搜索**：Reflection Tree（叶节点=基本观察，非叶节点=抽象反思，根=最高抽象）
  - **矢量检索**：向量数据库快速检索相关历史记忆

## 推理（Reasoning CoT）

- CoT 将计划、行动、观察结合，弥合推理与行动的差距
- AgentBench：强制"思考+行动"步骤
- 行动链技术：历史动作链+未来计划 → 决策问题转为 CoT 推理
- 工具使用：代码操作机器、API 获取数据、浏览器实时检索、计算工具
