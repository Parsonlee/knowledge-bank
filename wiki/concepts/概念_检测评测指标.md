# 概念：检测评测指标

## 指标

- **IoU（交并比）**：预测框与真实框的交集 / 并集，判定预测框是否正确的阈值标准。
- **Precision / Recall**：查准率与查全率的权衡（Type I / Type II error）。
- **AP（Average Precision）**：单类别在不同 Recall 水平的平均精度。
- **mAP（mean AP）**：所有类别 AP 的均值，为检测核心指标。
  - VOC 风格：单一固定 IoU 阈值（如 0.5 或 0.75）
  - COCO 风格：IoU 阈值 0.5→0.95 步长 0.05 取均值，另分小/中/大物体 AP
- **FPS（Frames Per Second）**：速度指标。两阶段通常单 GPU 个位数 FPS，单阶段可达数十 FPS。跨硬件难以标准化对比。
