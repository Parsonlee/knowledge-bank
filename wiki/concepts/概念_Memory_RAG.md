# 概念_Memory_RAG

> tags: RAG
> confidence: high

## 定义

Memory RAG 通过引入显式记忆机制，弥补传统 RAG 系统中隐式记忆和实时检索之间的空白。在长文档理解、个性化知识管理等场景中表现出色。

## 记忆类型

- **隐式记忆**：存储在模型参数中的知识（预训练权重），快速推理但难以更新
- **显式记忆**：压缩的、结构化的长期知识表示（如对整本书或用户行为模式的理解），比隐式记忆更灵活且易于更新
- **工作记忆**：临时存储的检索文本块，用于当前任务的生成，类似人类短期记忆

## 技术实现

- **Memory3**：两阶段预训练将原始输入转换为显式记忆（稀疏键值缓存）
- **MemoRAG**：轻量级 LLM 作为全局记忆系统，提升长文本处理能力
- **CAG**：预计算键值缓存完全消除实时检索需求，提高系统效率
