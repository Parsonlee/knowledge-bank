# 半可分离矩阵

Semiseparable Matrix，SSM 序列变换的代数结构，是 Mamba2 SSD 框架的数学基础。

## 定义

一个矩阵 M 是半可分离的，若：
1. 是下三角矩阵（M_ij = 0 当 i < j）
2. 下三角部分的每个子矩阵都是低秩的（秩 ≤ N）

## SSM 中的体现

SSM 的序列变换 Y = MX 中，M 的元素为：

```
M_ij = C_i^T · A_i · A_{i-1} · ... · A_{j+1} · B_j  （i ≥ j）
M_ij = 0                                               （i < j）
```

当 A_t 为标量时（1-SS 矩阵），M 退化为最简单的半可分离形式，此时与线性 Attention 完全等价。

## 为何重要

- 半可分离结构 → 可以用分块矩阵分解高效计算
- 对角块（小的半可分离矩阵）→ 类 Attention 二次计算
- 非对角低秩块 → 批量矩阵乘（batched matmul）
- 使 SSD 算法能充分利用 GPU 矩阵单元
