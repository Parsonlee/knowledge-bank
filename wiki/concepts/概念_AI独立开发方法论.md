# 概念_AI独立开发方法论

> 出处：[[AI时代如何做独立开发]]（confidence: high）

艾逗笔（idoubi）一年半独立开发经验总结的方法论框架。

## 五条核心感悟

1. **天下武功唯快不破**：快速上线先起飞再加油，验证需求再决定是否继续投入。AI 时代快是优势。
2. **快不如精，长期主义**：跑通 PMF 后应坚持做下去，浅尝辄止是最大教训。
3. **梦想要大，切入要小**：独立开发者/小团队从垂直赛道切入——竞争更小、粘性更高、更易正反馈。"先垂再通，农村包围城市"。
4. **自己造血，打铁还需自身硬**：对资本祛魅，能自己造血就不需杠杆。
5. **流量为王，持续构建影响力**：积极经营社交账号，输出高质量利他内容，形成流量池。

## AI 应用出海可复制 SOP

- 技术栈：TypeScript / React / Next.js + Supabase/Neon + Stripe + Cloudflare/Vercel。
- 一小时快速上线：全栈框架 + 项目模板/商业模板 + UI 组件库。
- ProductHunt 打榜：冷启动最快方式，争取被 Featured，发布当天号召投票。
- 程序化 SEO：数据采集→AI 摘要→SSR→长尾关键词自动构建页面→定时更新 sitemap。
- AI Wrapper 指南：追热点抢域名 → Vibe Coding + AI 能力 → SEO+社媒 → 订阅+广告变现。

## 四个 all in 方向（2025）

1. **AI Coding**：Editor/Copilot/Agent/Vertical Agent 四类，小团队从小场景切入。
2. **Agent**：最主流 AI 产品形态，用 Agent 改造原有 SaaS，从垂直场景切入。
3. **Agent Infra**：卖铲子方向——Tools/Planning/Memory/Boilerplate/VM/Auth/Payment。
4. **MCP 生态**：AI 三年来最大平台型机会——MCP Server / 应用市场 / 路由平台 / 消费终端。
