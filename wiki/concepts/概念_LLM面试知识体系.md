# 概念：LLM 面试知识体系

LLM 面试覆盖五大核心模块，以下是各模块的关键知识点清单。

## 核心架构

- **Tokenization / BPE**：文本→token，子词处理 OOV；参见 [[概念_BPE分词算法]]
- **注意力机制**：Q/K/V 向量、点积相似度、Softmax 归一化
- **多头注意力**：分头关注句法/语义不同方面；参见 [[概念_自注意力复杂度]]
- **位置编码**：正弦/学习向量，维持 token 顺序
- **残差连接 + 层归一化**：解决梯度消失问题
- **编码器 vs 解码器**：理解任务（BERT 双向）vs 生成任务（GPT 自回归）

## 微调与训练

- **LoRA / QLoRA**：低秩旁路微调，4-bit 量化；参见 [[概念_LoRA低秩适应微调]]
- **PEFT**：只更新少量参数，保留预训练知识
- **灾难性遗忘**：重播/EWC/模块化架构
- **模型蒸馏**：学生学习教师软概率，压缩部署
- **三阶段训练**：预训练→SFT→RLHF；参见 [[概念_大模型训练三阶段]]

## 推理生成

- **温度参数**：高温创造性，低温确定性；参见 [[概念_温度参数与幻觉创造力权衡]]
- **Top-k / Top-p**：采样候选截断策略
- **束搜索 Beam Search**：保留 k 个候选路径
- **CoT 思维链**：逐步推理提升复杂任务准确性；参见 [[概念_思维链CoT高级方法]]
- **RAG**：检索增强生成，提升事实准确性；参见 [[概念_RAG基础流程]]

## 扩展能力

- **MoE**：稀疏专家门控，10% 参数激活；参见 [[概念_MoE混合专家]]
- **零样本/少样本学习**：利用预训练通用知识适应新任务
- **知识图谱集成**：验证事实减少幻觉

## 数学基础

- **Softmax**：注意力分数→概率分布
- **交叉熵损失**：惩罚错误 token 预测
- **KL 散度**：衡量两分布差异，用于对齐微调
- **链式法则**：反向传播梯度计算基础
