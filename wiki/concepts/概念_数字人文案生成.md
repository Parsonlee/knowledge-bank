# 概念：数字人文案生成

淘宝直播数字人场景下，从 LLM 生成商品口播文案的完整技术体系。来源：[[淘宝直播数字人_LLM文案生成技术]]

## 核心特殊性

数字人文案生成与TTS强绑定，需优先保证文案"能被正确念出来"，同时追求口语化拟人效果。

## 四大技术模块

### 1. 基于语意的口播文案改写

解决 TTS 读错数字/英文/符号问题（如 88VIP 读"八八VIP"不读"八十八"）。

训练方案：
- Qwen2.5-7B 基座 SFT（人工精标 + DeepSeek-R1 难样本数据增强）+ DPO
- 通用数据增强（实验2）无效，必须针对性难样本增强（实验3 92%→95%）
- DPO 对比学习进一步提升至 97%（实验4）

### 2. 口语化改写（去除机械感）

依托 4000 小时真人 ASR 数据，DeepSeek-R1 蒸馏口语化改写模型。

关键发现：加入 CoT 思考过程（R1-Distill + 思考）评分 84，优于无思考版本（77/79）。

### 3. 多源信息文案生成（多步→单步蒸馏）

多源输入：商品详情 + 问评买 + 实时权益 + 素材理解 + 个性化人设（RAG 检索）

训练：多步 GPT4o 生成高质量数据 → 蒸馏到单步 Qwen2.5-7B + DPO

效果：满意度 78.86（接近多步 GPT4o 80.03），生成耗时从 13.52s 降至 3.41s（减少 75%）

### 4. 素材文案结合（图文一致）

OCR 提取图片文字 → kmeans聚类过滤+坐标排序 → BERT图片分类（87%准确率）→ LLM 生成对应讲解话术

素材选取策略：优先级别筛选+过滤字数过多/纯图少字/字体过小的图片

## 测评维度体系

- **60分及格**（基础能力）：格式化、口语化、可信度（幻觉检测）、安全性（极限词）、丰富度（distinct-k）
- **80分良好**（内容质量）：针对性卖点讲解、逼单话术购买欲
- **80分以上**：需融合人设/销冠真人主播风格，暂无固定标准

评估方法：规则匹配 + BERT 余弦相似度 + LLM 打分
