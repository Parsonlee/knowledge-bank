# 概念：Agent 仿脑区功能分区架构

**来源**：[[从代码生成到自主决策_Coding驱动的自我编程Agent]]

借鉴人类大脑分区思想，将 Agent 逻辑区域划分为五个功能区（Area），每个 Area 包含同类 Act，整体清晰且支持相互沟通。每个 Agent 是一个线程，Area 和 Act 在线程池中并发运行。

## 五大功能区

| 功能区 | 对应脑区 | 核心职责 |
|--------|---------|---------|
| 感知区 | 感觉皮层 | 接收外部信息（用户消息/卡片事件/子 Agent 返回）；解析主语言/模糊化/场景/附件 |
| 认知区（IntentPlanner） | 前额叶 | 接收感知解析结果，生成 Segment 并提交代码执行；复杂任务移交运动区 |
| 运动区（TaskExecutor） | 运动皮层/高级认知 | 承担复杂任务，内部循环逐轮执行计划，每轮评估进度/制定下轮目标 |
| 表达区 | 布罗卡区 | 向外传递信息（文本/卡片/事件）；文本不过度总结 |
| 自我评估区（SelfTaskCheck） | 元认知 | 执行后评估任务完成情况，失败则重新触发来源 |

## 运动区（TaskExecutor）工作方式

逐轮执行，每轮重新制定子目标，轮次 Segments 格式：
```
Round N: [历史所有Segments, ThoughtN, PlanN, CmdN, EvaluationN, NextGoalN]
```
任务完成 → 汇报感知区；无法完成 → 可选择放弃。
