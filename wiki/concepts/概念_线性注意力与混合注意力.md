# 概念：线性注意力与混合注意力

## Linear Attention

### 核心原理
- 标准 Softmax Attention 复杂度：O(N²)（N 为序列长度）
- Linear Attention 将 softmax(QK^T)V 改写为 φ(Q)(φ(K)^T V)，利用结合律先算 K^T V
- 复杂度降为 O(N)，推理时等价于循环神经网络，不需要 KV Cache

### 工程挑战（MiniMax Haohai 总结）
1. **工程链路复杂性爆炸**：多场景同时支持，验证成本高
2. **评测体系局限**：小模型结论不能外推，复杂多跳推理缺陷在大规模才暴露
3. **基建不完善**：训练访存 bound，推理缺乏低精度存储/Prefix Cache/投机解码成熟方案

### 代表方案
- GDN（Gated DeltaNet）
- Mamba/Mamba2（SSM）
- KDA（Kimi Delta Attention）

## Hybrid Attention（混合注意力）

### 核心思路
- 混合使用 Linear Attention 层和 Full Attention 层
- 以 Full Attention 弥补 Linear Attention 的性能短板

### 关键参数：混合比例
- Kimi Linear 通过 ablation study 确定最优 **3:1**（KDA:MLA）
  - PPL 对比：纯 MLA=5.77，3:1=5.65（最优），1:1=5.66，7:1=5.70

### 代表方案
- Kimi Linear：KDA + MLA = 3:1
- 腾讯混元 TurboS：Transformer-Mamba 混合

## 行业现状（2025年10月）

| 公司 | 方案 |

| MiniMax M2 | Full Attention（回归） |
