# 概念：AI Agent 四种类型

基于自主性和适用场景，AI Agent 可分为四种主要类型。

## 四种类型

### 1. 反应型 Agent（Reactive Agent）

- **特点**：基于当前状况快速做出决策，不需要复杂推理
- **类比**：经验丰富的专家即时反应
- **适用**：高频交易系统、游戏 AI、工业控制系统

### 2. 目标导向型 Agent（Goal-oriented Agent）

- **特点**：制定并执行长期策略，分解目标为子任务，监控进展调整策略
- **类比**：制定完整营销策略的智能助手
- **适用**：智能营销、项目管理、复杂规划

### 3. 学习型 Agent（Learning Agent）

- **特点**：从历史数据总结经验，通过试错优化策略，适应环境变化
- **类比**：能持续进化的客服系统
- **适用**：个性化推荐、持续优化的对话系统

### 4. 协作型 Agent（Collaborative Agent）

- **特点**：具备通信协调能力，理解配合其他 Agent，动态调整角色
- **类比**：分布式团队协作
- **适用**：智能物流（路线/库存/配送多 Agent 协作）、多 Agent 工作流

## 现代 Agent 特点

随着技术发展，越来越多 Agent 具备多种类型特征，能更全面满足复杂场景需求。
