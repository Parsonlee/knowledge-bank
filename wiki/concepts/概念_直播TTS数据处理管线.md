# 直播 TTS 数据处理管线

[[实体_淘天AIGC团队]] 为数字人直播 TTS 搭建的完整数据处理流程，从直播回放数据中提取高质量训练语料。

## 三核心环节

### 1. 语音信号处理
- **归一化**：统一不同分布下音频采样率/响度
- **背景声&说话人分离&去噪**：UVR_MDXNET 人声伴奏分离 → Resemble Enhance 提质 → Speaker Diarization 区分说话人 → 多说话人重叠检测过滤
- **VAD/停顿截断**：VAD 判人声起止点 + 语音能量特征细粒度静音检测，裁剪过长停顿
- **质量分过滤**：基于 DNSMOS 分/时长过滤低质量音频

### 2. 语音理解（文本标注）
- **ASR**：Seaco-Paraformer（热词微调）+ Whisper-large-v3-turbo 交叉验证提高置信度
- 迭代方式：少量标注+伪标签+合成数据→多轮迭代优化
- **标点修复**：结合 ASR 标点和音频未发声片段修复多打/少打
- **拖音/停顿韵律标记**：基于字尾时长+短时能量特征区分标记

### 3. 说话人聚类
- 提取语音 embedding → 余弦相似度 + 无监督聚类
- 时长过短簇删除 → 数据优选策略选取优质数据和参考 prompt

## 设计理念

- 一层层"漏斗"逐步提高最终训练数据质量
- 利用真实直播回放替代传统录制（成本低/分布丰富/更有直播感）
- 数据处理&模型训练解耦，定时处理最新直播回放
