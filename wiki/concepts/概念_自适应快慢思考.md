# 概念：自适应快慢思考（Adaptive Reasoning）

## 问题背景

Large Reasoning Model（LRM）具备 LongCoT 能力（Inference Time Scaling），但长思考 = 高延迟 + 高成本。并非所有问题都需要长 CoT：简单问题可快速回答，复杂问题才需"打草稿"。

## 三种方案

### Qwen3 混合思考（SFT 驱动，人为控制）
- 构造 `/think` / `/no_think` 混合 SFT 数据，RL 阶段加 format 奖励
- 需人为在 prompt 中添加标签指示模型是否思考
- 涌现 Thinking Budget：接近 max_tokens 时可触发强制终止思考

### AdaCoT（字节 Seed，模型自主 + Pareto PPO）
- 优化目标：最小化 CoT 触发次数，同时最大化模型得分（Pareto 优化）
- PPO 奖励：R_base - α1·P_miss - α2·P_over - γ·P_fmt
- **Selective Loss Masking**：mask `<think>` 后第一个 token 的 loss，防止模型探索坍缩
- 结果：65% CoT 触发率 ≈ 100% 触发效果；实际用户查询 3.18% 需要 CoT

### AdaThinking（清华，模型自主 + 约束优化 PPO）
- 优化目标：最大化不思考率，约束新模型 reward ≥ 参考模型 reward
- 拉格朗日转化 → PPO 优化
- **修改重要性采样**：初始 thinking model 的 y₁=`</think>` 概率为 0，通过修改 π_IS 使模型能探索"不思考"
- 更新条件：不思考得分 + δ > 参考得分 且 > 思考模式得分

## 关键洞察

- 简单问题 CoT 无益甚至有害；越难的问题越需要更长思考
- 只有 Level 5 难度时 Thinking 才明显优于 Non-Thinking（AdaThinking 分析）
- RL（尤其是 PPO）是实现模型自主决策是否思考的核心技术路线
