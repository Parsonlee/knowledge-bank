# 概念：通信计算重叠

## 定义

分布式 GPU 集群训练中，常规训练时序是串联式的，存在许多网络 IO 导致 GPU 利用率低。通信计算重叠是 AI Infra 提升 GPU 利用率的方法论：希望 GPU 大部分时间都在计算，而非花在数据传输或等待其他 GPU 上。

## 原理

GPU 编程模型有流（stream）概念：一个流表示一个 GPU 操作队列，队列内操作按添加顺序依次执行，不同流之间可并行执行。通过令计算和通信操作加入不同的流，可做到二者执行在时间上重叠。

## 类比

传统后台服务通过多线程或异步 IO 避免阻塞 CPU 主线程。

## 实现

例如 TorchRec 的训练流水线可实现高效的通信计算重叠。
