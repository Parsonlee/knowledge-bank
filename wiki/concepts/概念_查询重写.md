# 概念_查询重写

> tags: RAG, RAG/query
> confidence: high

## 定义

查询重写（Query Rewrite）是一种 Query Translation 方法，通过 LLM 将用户原始查询重新表述为更具体、更详细的查询，提高检索相关信息的可能性。重写后的查询不仅与原始查询语义相似，还提供不同角度或视角。

## 目的

- 使模糊查询更具体和详细
- 提供不同角度的表述
- 提高最终生成的质量和深度

## 示例

- 原始："气候变化对环境的影响是什么?"
- 重写："气候变化对各种生态系统的具体影响是什么，包括温度、降水模式、海平面和生物多样性的变化?"

## 实现

使用 Prompt 指导 LLM 重新表述查询：
```
You are an AI assistant tasked with reformulating user queries to improve retrieval in a RAG system.
Given the original query, rewrite it to be more specific, detailed, and likely to retrieve relevant information.
```
