# 概念：AI Workflow 与 Agent 对比

AI Workflow 和 AI Agent 是 LLM 时代两种主要的应用范式，核心区别在于**自主性与结构化程度的权衡**。

## 核心对比

| 维度 | AI Agent | AI Workflow |
|------|----------|-------------|
| 决策方式 | LLM 动态规划，自主决策 | 人类预定义流程，固定执行路径 |
| 自主性 | 高，能处理不确定性 | 低，结果可预测 |
| 灵活性 | 强，动态调整策略 | 弱，标准化执行 |
| 适用场景 | 复杂、动态、非结构化任务 | 重复、结构化、标准化任务 |
| 可控性 | 低 | 高 |
| 典型用例 | 智能客服、自动驾驶、金融交易 | 企业流程自动化、数据处理、文档归档 |

## Workflow 工作生命周期

1. 任务分解与流程设计（模块化）
2. 规则引擎构建（决策树）
3. 智能流程编排（节点逻辑）
4. 异常处理机制（多层容错）
5. 持续监控与优化

## 选型原则

- **高可控、标准化场景** → Workflow
- **灵活应变、复杂推理场景** → Agent
- **生产环境** → 往往需要 Workflow 的确定性 + Agent 的灵活性混合
