# 文生视频提示词公式

[[实体_通义万相Wan2.2]] 文生视频提示词的结构化公式体系。

## 三套公式

### 基础公式
提示词 = 主体 + 场景 + 运动
- 主体：视频内容主要表现对象
- 场景：主体所处环境
- 运动：主体具体运动和非主体运动状态

### 进阶公式
提示词 = 主体描述 + 场景描述 + 运动描述 + 美学控制 + 风格化
- 主体描述：外观特征细节
- 场景描述：环境特征细节
- 运动描述：幅度/速率/作用效果
- 美学控制：光源/光线/景别/视角/镜头/运镜
- 风格化：画面风格语言

### 图生视频公式
提示词 = 运动 + 运镜（图像已确定主体/场景/风格）

## 关键规律

- 描述越完整精确丰富，生成视频品质越高
- 进阶公式中美学控制项（光线/景别/构图）对画面质感影响最大
- 图生视频时运动/运镜描述优先级高于静态构图描述
