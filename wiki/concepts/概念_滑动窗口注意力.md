# 概念_滑动窗口注意力

滑动窗口注意力（Sliding Window Attention，又称 Local Attention）是一种局部注意力机制，限制每个 Token 只能关注其周围固定窗口内的上下文，而非全序列。

## 原理

- 常规自注意力（全局注意力）：每个 Token 可访问序列中所有其他 Token
- 滑动窗口注意力：每个 Token 只关注前后各 w/2 个 Token（窗口大小 w）
- KV Cache 只需存储窗口内的 K/V，内存从 O(N) 降为 O(w)

## 与全局注意力的混合使用

实践中通常混合使用局部和全局注意力：

- **Gemma 2**：全局:局部 = 1:1，窗口大小 4096
- **Gemma 3**：全局:局部 = 1:5，窗口大小缩小到 1024，内存节省更大
- 消融研究：混合比例调整对模型 perplexity（困惑度）几乎无影响

## 优缺点

- 优点：大幅降低 KV Cache 内存需求，适合长序列推理
- 缺点：不直接降低推理延迟（Mistral Small 3.1 因此放弃滑动窗口，改用 FlashAttention）
