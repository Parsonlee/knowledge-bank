# 概念_文档结构切分

> tags: RAG, RAG/chunking
> confidence: high

## 定义

文本切分五层级中的 Level 3。让切分策略适配不同的数据格式（JSON、Markdown、代码、PDF 等），利用文档自身结构进行切分。

## JSON 切分

- 遍历 JSON 数据深度，构建更小的 JSON 块
- 尽量保持嵌套 JSON 对象完整，在需要时拆分以控制块大小
- 大字符串值不会被自动分割（需要时组合递归文本分割器）
- **列表问题**：大列表会被完整保留导致片段超长
- **解决方案**：`convert_lists=True` 将列表转换为 `{index: item}` 字典形式

## 实现

- LangChain：`RecursiveJsonSplitter(max_chunk_size, min_chunk_size)`
  - `split_json(json_data, convert_lists=True/False)`
- LLama-Index：`JSONNodeParser()` + `get_nodes_from_documents([Document(text=json.dumps(data))])`
