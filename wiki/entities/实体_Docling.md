## 简介

Docling 是 IBM 开源的 PDF 解析库，在企业年报 RAG 挑战赛中是表现最好的解析器（IBM 也是比赛协办方之一）。

## 特点

- 解析质量优秀，但缺乏一些基本能力（功能分散在不同配置中无法组合）
- 可通过 GPU 加速解析（挑战赛中租 4090 虚拟机，100 份文档约 40 分钟，速度极高）
- 表格解析能力足够好，实测无需额外表格序列化辅助

## 实践经验（RAG 挑战赛）

- 重写其源码方法，导出含完整元数据的 JSON
- 基于 JSON 构建 Markdown，表格从 PDF 转 Markdown 甚至 HTML，还原度近完美
