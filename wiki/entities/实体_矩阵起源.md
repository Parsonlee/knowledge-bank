# 实体：矩阵起源（MatrixOrigin）

矩阵起源（MatrixOrigin）是 MatrixOne 数据库和 Matrix Intelligence 平台的开发商，专注于 AI-ready 数据基础设施。

## 主要产品

### MatrixOne（数据库）

- 类型：HTAP（Hybrid Transactional and Analytical Processing）数据库
- 优势：避免"元数据在交易库、数据在分析库"的割裂状态
- 特性：统一整合 Schema/业务数据/日志，为 AI-ready data 提供单一可信源
- 内建向量引擎，支持语义检索和相似度计算

### Matrix Intelligence（平台）

- 功能：统一纳管 Schema/元数据/业务词典/权限
- 形成面向 AI 的数据资产目录
- 提供可调度/可监控的 SQL 数据工作流
- 辅助发现同义词/识别命名不规范字段（影子字段）

## 在 NL2SQL 中的角色

作为 NL2SQL 工程化路径中"统一数据底座"的技术选型：
- 从源头解决语义缺席问题
- 将数据准备从手工脚本升级为可治理/可复用的工程化体系
- 实现 AI-ready data 的规模化生产
