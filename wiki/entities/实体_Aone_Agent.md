# 实体：Aone Agent（阿里通用 Agent）

阿里巴巴代码平台开发的通用 Agent 产品，类 Devin，目标实现全研发全生命周期提效。

- **架构**：Leader Agent 分发任务 → 子 Agent 执行（terminal/shell/文件/Web/MCP），容器内运行
- **能力**：技术/业务调研、需求代码生成、前端代码 + 调试 + 预览
- **工作流程**：用户需求 → 后台启动容器 → 运行引擎 → 生成代码 → 容器内测试 → 反思循环
- **目标**：异步完成复杂任务，潜在 10 倍效率提升

核心挑战：记忆管理（短期/长期）、长链路评估、工具间通信、指令跟随、高成本（Claude 处理一个任务约 50-80 元）。
