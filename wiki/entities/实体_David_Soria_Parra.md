## 简介

David Soria Parra 是 Anthropic 工程团队成员，MCP（Model Context Protocol）的联合创建者之一。

## 贡献

- 联合设计 MCP 协议，包括五大原语（Tool/Prompt/Resource/Sampling/Roots）
- 主导 MCP Web 化方向：Streamable HTTP 传输标准与 OAuth 2.1 鉴权方案
- 公开阐述 MCP 的正确使用姿势："绝大多数人用法都太初级"（仅使用 Tool）

## 核心观点

- MCP 的三类原语（Prompt/Resource/Tool）分别对应不同触发主体（用户/应用/模型），需协调使用才能构建完善的 AI 应用体验
- Sampling 原语支持链式调用（Chain of MCP Servers），是目前最欠利用的原语
- MCP 的未来在 Web：本地 1 万+ MCP Server 只是起点，最终形态是变成网站
