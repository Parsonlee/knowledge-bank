# 实体：AutoGPT

AutoGPT 是 2023 年 3 月推出的早期 AI Agent 代表项目，由 Significant Gravitas Ltd. 的创始人 Toran Bruce Richards 开发。

## 基本信息

- GitHub：https://github.com/Significant-Gravitas/AutoGPT
- 发布时间：2023 年 3 月
- 状态：GitHub 上最受欢迎的 Agent 项目之一

## 核心特点

- 使用 OpenAI GPT-4 和 GPT-3.5 大语言模型
- 完整工具包，用于构建和运行自定义 AI Agent
- 支持各种个人和商业项目的 Agent 构建
- 采用自我引导循环方式工作

## 局限与问题

社区多次反馈的典型问题（来自 Reddit 早期帖子）：
- **循环/卡死**：无法完结任务
- **幻觉严重**：多步任务中跳步/幻觉明显
- **不可用于生产**：仅适合 Demo 场景

AutoGPT 是"能跑但不可靠"的典型案例，验证了 Agent 从 Demo 到生产的工程复杂性。
