# 实体：AutoGen

AutoGen 是微软开源的多智能体对话框架，支持多个 Agent 相互对话协作来解决任务。

## 基本信息

- GitHub：https://github.com/microsoft/autogen
- 开发方：微软
- 定位：多 Agent 对话框架

## 核心特点

- **多 Agent 对话**：Agent 可相互对话解决任务
- **可对话、可定制**：Agent 灵活配置
- **支持人类参与**：无缝允许 Human-in-the-loop
- **增强 LLM 推理**：API 统一/缓存/错误处理/多配置推理

## 支持的对话模式

- LLM + 人工输入组合
- LLM + 工具使用组合
- LLM + 人工输入 + 工具综合使用
