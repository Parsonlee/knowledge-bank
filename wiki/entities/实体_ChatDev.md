# 实体：ChatDev

ChatDev 是一家虚拟软件公司，通过多个担任不同角色的 AI Agent 协作完成软件开发任务。

## 基本信息

- GitHub：https://github.com/OpenBMB/ChatDev
- 开发方：OpenBMB（清华/面壁智能）
- 定位：多 Agent 协作的虚拟软件公司

## 角色体系

- 首席执行官（CEO）
- 首席产品官（CPO）
- 首席技术官（CTO）
- 程序员
- 审阅者
- 测试员
- 美术设计师

## 协作模式

Agent 通过参加专门功能研讨会进行协作：
- 设计研讨会
- 编码研讨会
- 测试研讨会
- 文档研讨会

## 特点

- 提供易于使用、高度可定制和可扩展的 LLM 框架
- 是研究集体智慧的理想场景
- 使命：通过编程彻底改变数字世界
