# 实体：MetaGPT

MetaGPT 是一个开源 AI Agent 框架，模仿传统软件公司结构，通过角色扮演实现多 Agent 协作。

## 基本信息

- GitHub：https://github.com/geekan/MetaGPT
- 开发方：开源社区（geekan）
- 定位：软件开发多 Agent 框架

## 核心特点

- **角色体系**：产品经理/项目经理/工程师等角色
- **协作编码**：多 Agent 在用户定义任务上协作
- **适合中等难度任务**：蛇游戏/简单实用应用程序
- **成本参考**：完整项目约 $2 OpenAI 接口费用（2023 年）

## 与 ChatDev 对比

| 维度 | MetaGPT | ChatDev |
|------|---------|---------|
| 角色 | 产品经理/PM/工程师 | CEO/CPO/CTO/程序员/审阅者/测试员 |
| 定位 | 代码生成为主 | 虚拟软件公司 |
| 研究目标 | 工程化输出 | 研究集体智慧场景 |
