## 简介

MCP.so 是由艾逗笔（idoubi）于 2024 年 11 月上线的 MCP 应用市场，面向 Agent 开发提供 MCP Server 发现与分发。

## 成绩（全文所述）

- 最近三个月平均月访问量百万。
- 被 a16z 市场报告引用，已有全球知名度。
- 靠**程序化 SEO** 拿到 Google "MCP" 关键词搜索第一。

## 定位

- 对 MCP 服务器的管理和分发是核心需求（MCP 服务器数量急速增长）。
- 目标：做一个"AI 时代的豌豆荚"。
- 属于 MCP 生态的四大方向之一（MCP Server / 应用市场 / 路由平台 / 消费终端）。
