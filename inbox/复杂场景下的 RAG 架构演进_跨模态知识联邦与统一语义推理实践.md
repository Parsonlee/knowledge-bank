# 复杂场景下的 RAG 架构演进：跨模态知识联邦与统一语义推理实践

> [!quote]
> 融合知识库它主要是将左边散落的各种多元异构数据，包括指标、结构化数据、文档、图片等信息，通过知识融合技术整合到一个统一的融合知识库中。知识融合的过程涉及多种常用技术手段。

> [!quote]
> 从技术形态和产品形态来看，融合知识库是一个逻辑概念。底层的物理存储保持不变，例如文档仍然存储在向量数据库中，关系型数据库的数据源（如 MySQL 或 PostgreSQL）仍然保留在原处，只是它们的元数据被抽取出来，以逻辑视图的形式存储在融合知识库中。
